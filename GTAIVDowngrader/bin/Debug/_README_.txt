Thank you for using my automatic GTA IV Downgrader!
Begin by starting the GTAIVDowngrader.exe!

- - - Here are some useful tips for you - - -

- Upgrading to a new version -
- - - - - - - - - - - - - - - -
If you're upgrading from an older version of the GTA IV Downgrader, please delete the old version to avoid any confusions.

- Installing a hotfix -
- - - - - - - - - - - -
If there is a hotfix for the current version of the downgrader available download the hotfix and replace the current GTAIVDowngrader.exe with the one you've downloaded from the hotfix.
You should never run the GTAIVDowngrader.exe standalone.
The downgrader directory should always look something like this: https://i.imgur.com/a0a96TE.png
